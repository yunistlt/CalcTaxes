# Открытый финансовый калькулятор [![Hits](https://komarev.com/ghpvc/?username=yunistlt-calctaxes&color=58a6ff&style=flat-square&label=просмотров)](https://github.com/yunistlt/CalcTaxes)

## О проекте
Данный проект представляет собой бесплатное веб-приложение для проведения финансовых расчетов и построения базовых финансовых моделей. Сервис создан для того, чтобы дать пользователям простой, доступный и безопасный инструмент для экономического планирования.

## Ключевые особенности

- **Свободный доступ:** Сервис полностью бесплатен. Любой желающий может начать пользоваться калькулятором моментально, без прохождения процедуры регистрации.
- **Абсолютная анонимность:** Система автоматически создает индивидуальный аккаунт для каждого посетителя (на основе сессии/анонимного идентификатора). Мы принципиально **не запрашиваем, не собираем и не храним** никакие персональные данные пользователей.
- **Облачное сохранение данных:** Все производимые расчеты и созданные финансовые модели надежно сохраняются в нашей базе данных и привязываются к вашему анонимному аккаунту. Вы не потеряете свои наработки при обновлении страницы.
- **Экспорт результатов:** Реализована функция скачивания готовых расчетов. Пользователь может в один клик сохранить свою финансовую модель себе на устройство в формате PDF для дальнейшего использования, презентации или печати.

## Для кого этот сервис?
Инструмент отлично подойдет предпринимателям, самозанятым, финансовым аналитикам и всем, кому необходимо быстро прикинуть юнит-экономику, рассчитать налоги, фонд оплаты труда или построить финансовую модель проекта без необходимости разбираться в сложных таблицах.

## Принципы работы с данными
1. **Конфиденциальность:** Отсутствие имен, email-адресов или телефонов в нашей базе. Вы остаетесь полностью инкогнито.
2. **Доступность:** Данные расчетов доступны только вам в рамках вашей текущей сессии/анонимного профиля.
3. **Удобство:** Перенос сложных расчетов в понятный, читаемый PDF-документ.
